# Omnia Wellness App

CS 425 / 426 Senior Capstone Project – University of Nevada, Reno

## Overview
Omnia is a mobile wellness application designed to help users track and reflect on their health and wellness data in one place. The app combines automatic health data collection with manual wellness logs to support healthier habits and provide meaningful insights over time.

## Features
- Apple HealthKit integration
- Automatic step tracking
- Automatic sleep tracking
- Manual mood and stress logging
- Workout logging
- Nutrition tracking
- Goals and habit support
- Historical data views
- Insight generation based on user wellness trends

## Tech Stack
- React Native
- Expo
- TypeScript
- Supabase
- Apple HealthKit
